# Paradise Nursery

Paradise Nursery is an online plant shopping application built with React and Redux. Browse a wide selection of air purifying, aromatic, medicinal, insect repellent, and low maintenance plants. Add your favorites to the cart and enjoy a seamless shopping experience.

## Features
- Browse plants organized by category
- Add to cart with real-time updates
- Adjust quantity or remove items from cart
- Fully responsive design

## Tech Stack
- React + Vite
- Redux Toolkit
- GitHub Pages
